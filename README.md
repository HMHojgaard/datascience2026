# datascience2026
Repo for the Data Science course. Project focused on forecasting and prediction using NYC Taxi Trip data.

## Project Structure

### `forecasting_data/`
Contains processed datasets and time series data used for forecasting models.

### `forecasting_plots/`
Stores visualizations and figures related to forecasting analysis and model evaluation.

### `trip_lvl_prediction/`
Files and experiments related to trip-level prediction tasks.

## Notebooks

### `1_preprocessing_forecasting_data.ipynb`
Data cleaning, preprocessing, feature engineering, and preparation of forecasting datasets.

### `2_forecasting.ipynb`
Time series forecasting models, evaluation, and analysis.

## Dataset

This project uses the New York City Taxi & Limousine Commission (TLC) Trip Record Data.
https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page
